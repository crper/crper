# Hi, I'm crper

喜欢把重复劳动做成工具，把零散经验沉淀成可复用的东西。

平时主要折腾这些方向：
- AI 工作流和 Agent 工具
- 浏览器扩展
- 本地优先的小工具和 CLI
- 一些能立刻解决真实问题的开发者工具

## 🚀 当前项目

**AI 工作流**
- 🧠 [agent-skills](https://github.com/crper/agent-skills) · Skills 仓库 — 沉淀 Codex、OpenClaw 和自动化工作流里的可复用技能。
- 📚 [knowledge-chatbox](https://github.com/crper/knowledge-chatbox) · 本地优先的知识工作台，覆盖资料入库、检索问答、来源回看、Provider 配置与单机部署。
- 🖼️ [py-image-compress-mcp](https://github.com/crper/py-image-compress-mcp) · MCP 工具 — 为 AI Agent 提供本地图片压缩、格式转换和批量处理能力。

**浏览器扩展**
- 🧪 [参数工坊 · QueryCraft](https://chromewebstore.google.com/detail/%E5%8F%82%E6%95%B0%E5%B7%A5%E5%9D%8A-%C2%B7-querycraft/pnmnfenconpbfomddfhibbcnncmopgnb) · 面向开发和测试的 URL 参数编辑工具，支持解析、排序和二维码生成。
- 🧮 [股票收益计算器](https://chromewebstore.google.com/detail/%E8%82%A1%E7%A5%A8%E6%94%B6%E7%9B%8A%E8%AE%A1%E7%AE%97%E5%99%A8/moojoeabbjcccmmneljlcdlhediahfbe) · 提供连板收益、回本涨幅、定投等常用计算。
- 🛍️ [小店经营盘](https://chromewebstore.google.com/detail/%E5%B0%8F%E5%BA%97%E7%BB%8F%E8%90%A5%E7%9B%98/glehokfhfgiodnfhefeljndfbaijljlg?authuser=0&hl=zh-CN) · 面向小店经营场景的浏览器辅助工具。
- ♻️ [Auto Refresh](https://chromewebstore.google.com/detail/auto-refresh/npeboaflpngjenagipdkcepihphchcig) · 定时自动刷新网页，支持 URL 规则匹配，适合监控和轮询。
- 🔳 [wxt-chrome-qrcode](https://github.com/crper/wxt-chrome-qrcode) · 基于 WXT 的二维码生成扩展，支持样式调整、Logo 添加和多格式导出。

**桌面与在线工具**
- 📱 [tqrx](https://github.com/crper/tqrx) · 终端优先的二维码工具，支持 CLI / TUI 工作流、实时预览和 PNG / SVG 导出。
- 🔄 [py-local-git-pull](https://github.com/crper/py-local-git-pull) · 纯 CLI 的 Git 任务控制台，支持多仓库扫描、同步、诊断和运行历史。
- 🌍 [Time Shift](https://crper.github.io/next-time-shift/) · 快速完成时差换算与时间偏移，适合跨时区沟通和排期。（[Repo](https://github.com/crper/next-time-shift)）
- 📈 [Stock Return Calc](https://crper.github.io/next-stock-return-calc/) · 快速测算股票收益、涨跌影响和回本空间。（[Repo](https://github.com/crper/next-stock-return-calc)）
- 💹 [Simple Compound](https://crper.github.io/next-simple-compound/) · 快速估算复利收益，适合长期投资和目标测算。（[Repo](https://github.com/crper/next-simple-compound)）
- 🔷 [next-css-polygon-editor](https://crper.github.io/next-css-polygon-editor/) · 可视化编辑 CSS `clip-path` 多边形，拖拽后即可复制代码。（[Repo](https://github.com/crper/next-css-polygon-editor)）
- 🖥️ [Tauri-QrCode-BarCode](https://github.com/crper/Tauri-QrCode-BarCode) · 基于 Tauri + Vue 3 的离线二维码/条码生成器，支持导出图片。

**模板与脚手架**
- ✨ [next-modern-starter](https://github.com/crper/next-modern-starter) · Next.js 15 + React 19 + Tailwind CSS v4 的极简启动模板。
- 🧩 [vscode-webview-vite-vue-boilerplate](https://github.com/crper/vscode-webview-vite-vue-boilerplate) · VS Code Webview 模板，集成 Vite、Vue 3、TypeScript、本地调试和 vsix 打包。
- 📦 [rollup-typescript-lib-boilerplate](https://github.com/crper/rollup-typescript-lib-boilerplate) · TypeScript 库模板，集成 Rollup、Vitest、release-it 和常用提交规范。

**npm 工具库**
- 🧱 [placefill](https://github.com/crper/placefill) · 轻量占位符填充库，支持 `string / object / array` 的深度遍历与模板填充。
- 🎯 [tiny-event-intercept](https://github.com/crper/tiny-event-intercept) · 轻量、零依赖的条件事件拦截工具，兼容浏览器标准 API 和 SSR。
- ⚡ [fast-map-cache](https://github.com/crper/fast-map-cache) · 面向 Node.js 和浏览器的高性能 LRU 缓存，支持可选 TTL。
- 💤 [idle-resource-loader](https://github.com/crper/idle-resource-loader) · 利用浏览器空闲时间做资源预加载，减少主线程阻塞。

**其他工具**
- ☁️ [action-wttr-fetch-city](https://github.com/crper/action-wttr-fetch-city) · 用 GitHub Actions 抓天气并生成分享图片。
- 🌿 [vsc-git-branch-warning](https://github.com/crper/vsc-git-branch-warning) · VS Code 敏感分支提醒扩展，为 `main/master/test` 等分支提供状态栏和悬浮提示。
- ⌨️ [Visual-Studio-Code-CNKEYMAP](https://github.com/crper/Visual-Studio-Code-CNKEYMAP) · VS Code 快捷键中文映射，适合快速对照和查阅。
- 🐳 [yapi-docker](https://github.com/crper/yapi-docker) · 用 Docker 快速拉起 YApi 容器环境。

更多项目直接看 [GitHub Repositories](https://github.com/crper?tab=repositories)。
